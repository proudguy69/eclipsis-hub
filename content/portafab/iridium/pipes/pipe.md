## Description

The Pipe is an item that can transport iridium throughout networks. The pipe only has two openings, so we can only connect one item to another, therefore the pipe is used with junctions, which adds more openings. 

By selecting the pipe in the portafab, a blueprint for one of the pipe openings will appear, once you select where you want to place the first opening, the blueprint will extend out a pipe to a second opening, with a max length of 25 studs. The pipe is extremely flexible, but it does have limits, like a max length of 25 studs and I can’t turn more than 90deg. Because of this, there’s a lot of interesting strategies with pipes and their junctions.
